# Roadmap do Reflexo

Este documento descreve a visão de longo prazo e os planos futuros para o Reflexo e sua série de quebra-cabeças diários.

## 🎯 Visão Geral

Reflexo é o primeiro de uma série de **4 quebra-cabeças diários originais**, cada um focando em um tipo diferente de lógica:

1. **Reflexo** ✅ (Lançado - v1.0)
   - Simetria espelhada com exceções
   - Status: Completo e no ar

2. **Equilíbrio** (Em Desenvolvimento - v1.0)
   - Balanceamento lógico de pesos
   - Previsto: Q3 2026

3. **Mistura** (Planejado - v1.0)
   - Combinação de cores e padrões
   - Previsto: Q4 2026

4. **Trilha de Energia** (Planejado - v1.0)
   - Circuitos e fluxo de energia
   - Previsto: Q1 2027

## 🚀 Features Planejadas

### Para Reflexo (Próximas)

#### Curto Prazo (1-3 meses)
- [ ] **Sistema de Dicas**
  - Revelar uma célula aleatória
  - Limite de 2 dicas por dia
  - Dica premium: mostrar regra violada

- [ ] **Estatísticas Detalhadas**
  - Taxa de vitória geral
  - Tempo médio de conclusão
  - Tempo mínimo (recorde pessoal)
  - Gráfico de sequência temporal

- [ ] **Temas**
  - Tema Claro (padrão)
  - Tema Escuro
  - Tema Pastel
  - Customização de cores

#### Médio Prazo (3-6 meses)
- [ ] **Internacionalização (i18n)**
  - Inglês (EN)
  - Espanhol (ES)
  - Francês (FR)
  - Mais idiomas conforme demanda

- [ ] **Compartilhamento**
  - Compartilhar resultado no Twitter
  - Link de convite para amigos
  - Leaderboard público (opcional)

- [ ] **Acessibilidade**
  - Modo de alto contraste
  - Navegação 100% por teclado
  - Suporte a leitores de tela melhorado

#### Longo Prazo (6+ meses)
- [ ] **Sistema de Contas (Opcional)**
  - Login com Google/GitHub
  - Sincronizar progresso entre dispositivos
  - Backup na nuvem

- [ ] **Comunidade**
  - Forum de discussão
  - Desafios semanais
  - Torneios de speed-run

## 📱 Hub Central (Futura Arquitetura)

Quando todos os 4 jogos estiverem prontos, eles serão acessíveis através de um **hub central**:

```
jogo-puzzles.com/
├── /reflexo        (Simetria)
├── /equilibrio     (Balanceamento)
├── /mistura        (Cores)
├── /trilha         (Energia)
└── /dashboard      (Progresso geral)
```

### Dashboard Central
- Visualizar streak em todos os jogos
- Estatísticas cruzadas
- Próximos puzzles de cada jogo
- Configurações globais

## 💡 Ideias Exploradas (Baixa Prioridade)

Essas ideias estão em consideração, mas não são prioridade:

- [ ] **App Nativa**
  - PWA (Progressive Web App)
  - Offline-first com service workers
  - Push notifications para novos puzzles

- [ ] **Monetização** (se necessário)
  - Sem modelo de assinatura (jogos sempre gratuitos)
  - Possível: temas premium ou dicas premium
  - Possível: suporte voluntário (Patreon)

- [ ] **Análise e Pesquisa**
  - Coletar estatísticas anônimas (opt-in)
  - Pesquisar dificuldade relativa
  - Melhorar design com base em feedback

- [ ] **Customização de Puzzles**
  - Editor de puzzles dos usuários
  - Dificuldade customizável
  - Modo sandbox

## 📊 Métricas de Sucesso

Usaremos estas métricas para medir o sucesso:

1. **Engajamento**
   - Jogadores diários únicos
   - Taxa de retorno (Day 1, Day 7, Day 30)
   - Streak médio

2. **Qualidade**
   - Feedback dos usuários
   - Ratings/Reviews (se em loja)
   - Taxa de resolução

3. **Comunidade**
   - Contribuições no GitHub
   - Discussões e issues
   - Traduções contribuídas
